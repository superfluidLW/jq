Jinx - The Excel Java Add-In
=java("in excel")
------------------------------------------------------------------------------

Thank you for downloading Jinx.

Jinx is an Excel addin that allows you to write Excel worksheet functions,
menus and macros in Java.

To use Jinx you must install jinx.xll or jinx64.xll in Excel by going to
Options -> Add-Ins -> Manage Excel Addins in the Excel File menu. Use
jinx.xll if you are using a 32 bit verison of Excel, and jinx64.xll if
you are using a 64 bit version of Excel.

For up to date documentation please refer to https://exceljava.com.
For all enquiries or questions please email info@exceljava.com
